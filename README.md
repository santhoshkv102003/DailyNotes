### Installation

1. Clone the repository:
```bash
git clone https://github.com/santhoshkv102003/DailyNotes.git
```

2. Navigate to the project directory:
```bash
cd DailyNotes
```

3. Open `index.html` in your web browser to run the application locally.

## Usage ğŸ’¡

### Adding Expenses
1. Enter the amount in the "Amount" field
2. Add a description in the "Description" field
3. Click the "Add" button
4. View the updated total at the bottom

### Writing Daily Notes
1. Click on the text area under "Daily highlights"
2. Type your notes for the day
3. Click the "SAVE" button to store your notes

### Viewing History
- Click the "History" button at the bottom right to view your previous entries
